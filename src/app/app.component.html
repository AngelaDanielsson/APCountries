Branches = En kopia av orginalet. Man jobbar i branch för att slippa förstöra orginalet med kod som kanske inte är funktionell. cmd: git branch NAMN Exempel git branch nyttbranchnamn Nu måste man byta, då master står som aktiv. git checkout nyttbranchnamn Då cheackar man ut den man arbetar i och checkar in i den man skriver. Arbeta i den nya. När denna är fullt klar och fungerar. Spara den ctrl S. Pusha den till git, så kan alla se den. När alla är eniga, checkar man ut: git checkout master För att slå samman dessa: git merge nyttbranchnamn. Kör en git add . Sedan en git commit -m “Sammanslagning av nyttbrachnamn och master” Kör sedan en push igen.
