Branches

Branches = En kopia av orginalet. Man jobbar i branch för att slippa förstöra orginalet med kod som kanske inte är funktionell.
  1. cmd: git branch NAMN Exempel git branch nyttbranchnamn

    2. Nu måste man byta, då master står som aktiv.

  2. git checkout nyttbranchnamn Då cheackar man ut den man arbetar i och checkar in i den man skriver.
  3. Arbeta i den nya. När denna är fullt klar och fungerar. Spara den ctrl S.
  4. Pusha den till git, så kan alla se den.
  5. När alla är eniga, checkar man ut: git checkout master
  6. För att slå samman dessa: git merge nyttbranchnamn.
  7. Kör en git add .
  8. Sedan en git commit -m “Sammanslagning av nyttbrachnamn och master”
  9. Kör sedan en push igen.
